# 股票研究数据库 - ModelScope 部署

## 项目介绍
基于 Flask 的股票研究数据库 Web 应用，支持股票数据展示、概念分类、搜索等功能。

## 部署步骤

### 1. 创建创空间
1. 访问 https://www.modelscope.cn/
2. 登录后点击"创空间" → "创建创空间"
3. 选择"应用"类型
4. 填写名称和描述

### 2. 配置启动命令
在创空间设置中，配置启动命令：
```bash
python app.py
```

### 3. 端口设置
- 端口: 7860
- 类型: HTTP

### 4. 文件说明
- `app.py` - 主应用入口
- `requirements_modelscope.txt` - Python 依赖
- `templates/` - HTML 模板
- `data/master/stocks_master.json` - 股票数据

## 功能特性
- 📊 股票列表展示（支持分页）
- 🔍 股票搜索
- 🏷️ 概念分类
- 📈 股票详情
- 📥 数据导入（JSON格式）

## 数据说明
包含 3152 只 A 股股票数据，覆盖：
- 股票基本信息（名称、代码、板块）
- 行业分类
- 概念标签
- 相关文章

## 访问地址
部署成功后，访问创空间提供的 URL 即可使用。
